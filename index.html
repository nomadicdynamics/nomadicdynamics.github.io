Optimize Your Coding

Emmet uses common sense abbreviations to make code writing easier. Writing HTML code takes time with tags, attributes, quotes, braces, etc. Most text editors have code completion, that saves time, but you still have to do a lot of typing. Emmet expands simple abbreviations into complex code snippets.

Syntax

Getting started with a new HTML document takes a minimum of three keyboard presses. Just type ! or html:5, hit tab and you’ll see an HTML5 skeleton. Because Emmet’s syntax for describing elements is similar to CSS selectors, getting used to it is very easy. Curly brackets are used for content and square brackets are used for attributes.

♥

Analysis

Mixing an element’s name with an identifier will create a class or ID. The greater than symbol allows you to nest elements. The sibling operator is represented by + and lets you place elements on the same level. You can define how many times an element should be outputted by using the * operator.
